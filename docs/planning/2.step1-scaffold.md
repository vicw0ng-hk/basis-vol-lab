# Step 1: Scaffold Monorepo & Tooling

**Status**: ✅ Complete

## What was done

### Workspace structure

Created a uv workspace monorepo with the following layout:

```
basis-vol-lab/
  apps/
    web/          # Static front end (Cloudflare Pages)
    api/          # Thin dynamic endpoints (AWS Lambda)
    desktop/      # Optional PySide6 operator console
  packages/
    connectors/   # Async WebSocket/REST collectors
    analytics/    # Pandas/NumPy/SciPy analytics & signals
    contracts/    # Data models, schemas, enums
    persistence/  # SQLite/D1 metadata, Parquet/R2 time-series
  infra/
    terraform/
  benchmarks/
  tests/
  docs/
  notebooks/
```

Each `packages/*` and `apps/*` directory is a workspace member with its own `pyproject.toml`. Inter-package dependencies use `workspace = true` sources.

### Tooling configured

| Tool | Purpose |
|------|---------|
| `uv` | Python package/project management, workspace resolution |
| `mise` | Pin non-Python tools (node, terraform, wrangler) + project tasks |
| `ruff` | Format (Black-compatible) + lint (E, F, W, I, UP, B, SIM, N rules) |
| `pyright` | Static type checking |
| `pytest` | Test runner |
| `pre-commit` | Git hook enforcement of format/lint/typecheck/test |

### mise tasks

- `mise run format` — `uvx ruff format .`
- `mise run lint` — `uvx ruff check . --fix`
- `mise run typecheck` — `uvx pyright`
- `mise run test` — `uv run pytest`
- `mise run check` — runs all of the above in sequence

### CI (GitHub Actions)

`.github/workflows/ci.yml` runs on all pushes and PRs:
- **lint-and-typecheck** job: format check → ruff → pyright
- **test** job: uv sync → pytest

### Pre-commit hooks

`.pre-commit-config.yaml` enforces on every commit:
1. `ruff-format` — auto-format
2. `ruff` — lint with `--fix --exit-non-zero-on-fix`
3. `pyright` — type check
4. `pytest` — run test suite

Install with: `uv run pre-commit install`

### Dummy code for validation

- `packages/contracts` — `Venue`, `AssetKind`, `Currency` enums + `TickerSnapshot` dataclass
- `packages/analytics` — `annualized_carry()` function
- `tests/test_contracts.py` — 4 tests for models/enums
- `tests/test_analytics.py` — 5 tests for carry computation

All checks pass: format ✓, lint ✓, pyright 0 errors ✓, 9 tests pass ✓

## Commands reference

```bash
# Install/sync all workspace deps
uv sync

# Run full check pipeline
mise run check

# Or individually:
uvx ruff format .
uvx ruff check . --fix
uvx pyright
uv run pytest

# Install pre-commit hooks (one-time)
uv run pre-commit install
```
